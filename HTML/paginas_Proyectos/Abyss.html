Abyss

Un juego sencillo desarrollado inicialmente solo con Java, luego mejorado con JavaFX para una interfaz gráfica.

Desarrollo Inicial con Java

El primer paso fue crear Un Mapa funcional y la Movilidad del jugador. En este punto se habian creado ya muchas clases que simplemente existian para que otras heredasen de ellas

Mejoras con JavaFX

Luego, se incorporó JavaFX para mejorar la interfaz gráfica, lo que permitió una experiencia más atractiva y dinámica.

Pruebas y Ajustes

Se realizaron varias pruebas para ajustar la jugabilidad y asegurar que el rendimiento fuera adecuado en diferentes dispositivos.

Versión Final

Finalmente, se completó la versión final del juego, con todos los elementos de diseño y optimización para su lanzamiento.
